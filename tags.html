List of important HTML tags

Block elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people strong
a anchor make hyperlink <a href="http://reddit.com/">Reddit</a> Reddit
img image include an image <img src="https://upload.wikimedia.org/wikipedia/en/thumb/8/82/Reddit_logo_and_wordmark.svg/250px-Reddit_logo_and_wordmark.svg.png" alt="Reddit Icon">
code code Displays markup or source code <code>echo hello, world!<code> echo hello, world!
